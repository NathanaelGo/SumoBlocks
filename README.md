# SumoBlocks
A 3D Unity Game I made soon after I got into programing [2017]


The goal of this game is to push your opponent off the edge of the Sumo Square.
Outmaneuver or Overpower your way to VICTORY!




4 Unique Characters 

Jeff/Charles 

  -Basic Sumo Wrestler


Speedy

  -Fast and Small


Fatty

  -Bigger Slower and Stronger


Lanky

  -Longer model
